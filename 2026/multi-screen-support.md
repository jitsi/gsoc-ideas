# Multi-Screen support for Jitsi Meet

## Overview

Add support for displaying the conference on multiple screens.

## Description

Jitsi Meet currently assumes that one conference participant is represented by one browser window and one visual layout. This project would introduce native multi-screen support, allowing the same meeting session to be distributed across multiple windows and displays. A user could, for example, place the active speaker on one screen, gallery view on another, or dedicate separate displays to different participant groups or roles.

## Expected outcomes

* prototype implementation supporting two coordinated windows.
* support for assigning different layouts or participant selections to each window.
* synchronization logic between windows.
* handling for window open/close/reconnect scenarios.
* documentation and demo scenarios showing practical use cases.
